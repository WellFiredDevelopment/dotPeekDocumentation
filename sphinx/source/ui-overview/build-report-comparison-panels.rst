Build Report Comparison Panels
==============================

You can select several builds on the left side of the UI by pressing **Command** on MacOS or **Ctrl** on Windows while left clicking with the mouse, 
it will load a comparison of the newest and oldest selected build information, and display it through 4 panels :

1. :ref:`Overview Comparison <doc_ui-overview_build-report-comparison-overview>`
2. :ref:`Build Settings Comparison <doc_ui-overview_build-report-comparison-settings>`
3. :ref:`Unused Assets Comparison <doc_ui-overview_build-report-comparison-assets>`
4. :ref:`Used Assets Comparison <doc_ui-overview_build-report-comparison-assets>`
